# TP 1 - Dessin géométrique sur canvas

## Description

Interface web permettant de dessiner des formes géométriques sur un canvas HTML5.

### Fonctionnalités

- Dessiner des **lignes**, **rectangles** et **cercles**
- Modifier la taille du trait
- Changer la couleur de chaque forme

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/Orulirn/tpWe_1.git
   ```

2. Ouvrir `canvas.html` dans un navigateur

3. Amusez-vous !
